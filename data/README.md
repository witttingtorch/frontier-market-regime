# Data Directory

## Structure

| Folder | Purpose | Contents |
|--------|---------|----------|
| `raw/` | Original data from APIs | Never modify - source of truth |
| `processed/` | Transformed data | Cleaned, analyzed, ready for modeling |
| `logs/` | Human-readable outputs | Reports, classifications, summaries |

## Raw Data (`raw/`)

Downloaded directly from APIs:
- `cbk_kes_YYYY.csv` - Central Bank of Kenya FX rates
- `cba_azn_YYYY.csv` - Central Bank of Azerbaijan FX rates  
- `nse_stocks_YYYY.csv` - Nairobi Stock Exchange data

**Do not manually edit files in this folder.**

## Processed Data (`processed/`)

Generated by notebooks and pipeline:
- `*_regime_backtest.csv` - Historical regime classifications
- `*_eda.csv` - Exploratory analysis datasets
- `threshold_optimization.csv` - Grid search results
- `correlation_matrix.csv` - Variable correlations
- `optimal_thresholds.json` - Recommended thresholds
- `figure*.png` - Charts for paper/presentation

## Logs (`logs/`)

Human-readable outputs:
- `*_report.md` - Markdown summaries
- `regime_YYYYMMDD_HHMMSS.txt` - Individual classification logs
- `override_decisions.log` - Human override tracking
